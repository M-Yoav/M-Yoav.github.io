运行环境 Win10专业版
所需软件 windows terminal + powershell-7.2.3

# 安装软件

# Powershell

  1. 微软商店搜索 Powershell
  2. https://github.com/PowerShell 此链接会跳转到 powershell 的 github 仓库，自行选择版本

本文采用方法 2 ，下载完成后双击 .msi 程序，根据提示进行安装。

⚠️验证： win + R 输入 pwsh 能打开 powershell 终端或者输入 cmd 打开命令行窗口，输入 pwsh -v 出现版本号则安装成功。

# Windowns Terminal

✔️微软商店搜索 Windowns Terminal 并下载
⚠️验证： win + R 输入 wt 能打开 windowns terminal 终端说明安装成功
此时你的终端界面已经打败了系统原生自带命令行界面

# 安装主题

❤️我们需要给终端配个皮肤让他看起来更美观，那我们就需要一款免费开源的主题👉Oh-My-Posh

  • 安装 Oh-My-Posh，需要保证 Git 命令可用，这里默认 Git 已经安装，Git 安装可参考这篇文章
  • 使用管理员身份打开 Powershell。
  1. 第一条命令（安装 NuGet ），因为安装主题文件需要
  1. 第二条命令（设置存储库的安装策略）不懂可查看此文档
  1. 第三条命令（绕过 Powershell 执行策略，使其可以执行脚本文件）
  1. 第四条命令（安装 oh-my-posh） CurrentUser 表示当前用户设置 AllUser 表示为所有用户设置 -Verbose 可以显示安装过程
  • CurrentUser：安装后路径为 C:\Users\14875\Documents\PowerShell\Modules\oh-my-posh\7.85.2
  • AllUser： 安装后路径为 C:\Program Files\PowerShell\Modules\oh-my-posh\7.85.2 后面用得到
  • 使用命令安装后可在命令行输入 oh-my-posh --version , 出现版本号则安装成功
  1. 第五条命令（安装 posh-git）该模块会将 git 信息添加到提示中，例如 master~ 、 dev~

安装过程中出现提示，直接一路 Y 到底。

  1. 第六条命令（安装 Get-ChildItemColor）为 Powershell 的输出添加颜色，例如 ls

此时安装完成，在命令行输入 module 查看模块是否安装成功
